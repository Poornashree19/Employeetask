##  Setup Steps
1. Clone this repo  
   ```bash
   git clone https://github.com/Poornashree/employee-task-api.git
```
2. Install dependencies
```bash
cd backend
npm install
```

3. Start the server
```bash
npm start
```

API runs at: http://localhost:5000
